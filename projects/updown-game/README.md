# 🔢 Up & Down Game

> Python + Flet 기반 숫자 맞추기 게임

---

## 📌 Overview

1부터 100 사이의 랜덤 숫자를 맞추는 게임으로  
사용자의 입력에 따라 "업" 또는 "다운" 힌트를 제공합니다.

---

## 🚀 Features

- 랜덤 숫자 생성 (1~100)
- 사용자 입력 기반 비교
- 업 / 다운 힌트 제공
- 시도 횟수 카운트
- 게임 초기화 기능

---

## 🧠 Logic

- 랜덤 값(answer) 생성
- 사용자 입력값과 비교
- 상태값(attempts) 관리

---

## ⚙️ How to Run

```bash
pip install flet
python main.py
