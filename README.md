## 💡 Development Principles

1. **Atomization (세분화)**: 복잡한 문제는 해결 가능한 가장 작은 단위로 쪼개어 접근합니다.
2. **Zero Blind Code**: 코드 한 줄 한 줄의 의미를 완벽히 이해한 후 사용합니다.
3. **Assetizing Errors**: 오류 발생 시 원인과 해결 과정을 노션에 기록하여 자산화합니다.
